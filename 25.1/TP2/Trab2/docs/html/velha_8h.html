◆ verifica_velha()

int verifica_velha ( int jogo[3][3])

Recebe uma matriz de jogo da velha e retorna o número do jogador que venceu (1 ou 2), 0, -1 ou -2.

Parâmetros
jogoMatriz 3x3.
Retorna
0. Se o jogo está empatado, retorna 0.
1 ou 2. Se o jogo foi vencido, retorna o número do jogador que venceu.
-1. Se o jogo está indefinido (válido, mas não terminado), retorna 0.
-2. Se o jogo não é válido (viola as regras), retorna -2.
